A simple tap-based chase game where a Bird tries to catch a Pig on a scrolling forest path.

## Key Features

Tap / click based movement
Parallax scrolling background
Progress bar synced with player movement
Time-based win and lose conditions
Procedural animations (no Animator)
Clean, modular Unity architecture
Build for Pc (Build inside Build folder) 

## Unity Version
Unity 6.2
